wcc.donation Installation
-------------------------

To install wcc.donation using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.donation`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.donation

* Re-run buildout, e.g. with:

    $ ./bin/buildout
